# Armonía — Analizador Musical con IA

Herramienta web que analiza canciones musicalmente usando IA: tonalidad,
modos, funciones de cada acorde, modulaciones y mezclas modales.

## Features
- Análisis en 3 niveles: básico, intermedio y avanzado
- Soporte para ingresar acordes manualmente o via conocimiento del modelo
- Simulador de costos de API (Claude, Sonnet, Gemini, DeepSeek)
- Simulador de ingresos AdSense con P&L mensual y punto de equilibrio
- Streaming de respuestas en tiempo real

## Stack
- vite, shadcn-ui, tailwindcss v4
- Anthropic API (claude-haiku-4-5) para el análisis
- Google Fonts (DM Serif Display, DM Mono, Outfit)
